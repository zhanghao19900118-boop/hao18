# V3.1 发布说明

候选版本：`3.1.1-rc.1`

## 功能更新

- 用户拥有不可修改的唯一代码；管理员可以修改用户名，历史数据仍按用户代码对应。
- 增加世界杯届次模型和首页届次选择器，为后续世界杯扩展预留数据结构。
- 管理员可以下载 CSV 模板并批量导入初始积分预测记录。
- 管理员可以下载当前完整记录，在原文件上修改或新增；上传先预检查，确认后才写入，并且不会因文件缺行而删除数据库记录。
- 置信度统一为 `0–100.00`，最多保留两位小数。

## 修复

- 国旗改为随网站部署的 SVG 文件，不再依赖系统 emoji 或第三方图片服务。
- 比赛详情在缺少技术统计或球员数据时仍保留完整界面，以“待更新”明确标识缺失数据。

## 数据说明

当前赛程数据只包含赛程、比分、状态和场馆。控球率、射门、预期进球以及完整球员表现，需要后续导入对应数据源；本版本不会编造这些数值。

## 发布建议

先在 `feature/v3-auth-comments` 分支验证，通过后建立 Pull Request 合并到 `main`，最后创建 `v3.1.0` Release。
