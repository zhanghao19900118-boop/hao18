# 2026 DG 世界杯全纪录 V3.0

## 范围

- 管理员创建、停用、重置和禁言用户
- 登录、退出、首次登录强制改密
- 所有用户预测可见，仅本人可编辑未锁定记录
- 同场多条、50 字预测内容、1–100 整数权重
- 官方开赛时间后 3 小时锁定
- 比赛概览下方公开评论；登录用户可发，管理员可删/隐藏
- 整数积分与排行榜
- SQLite、审计日志、每日备份、健康检查

## 安全修正

确认版 PRD 中“密码直接明文保存”与同文档的安全条款冲突。V3 实现采用 bcrypt 密码哈希；数据库、日志和前端均不保存或返回明文密码。

## 发布策略

1. 在 `feature/v3-auth-comments` 分支完成评审与 CI。
2. 先在 Vultr 测试数据库和域名 HTTPS 环境预览。
3. 合并 `main` 后打 `v3.0.0` 标签。
4. 部署前运行数据库备份，失败时回滚代码和数据库。
